VAPID-KEY GENERATOR

⟫ WEB PUSH SETUP ⟪

Dieses Tool erzeugt einmalig die VAPID-Schlüssel für deine Lockscreen-Push-Benachrichtigungen. Die Generation läuft komplett im Browser — nichts wird an einen Server gesendet.

mailto:Kontakt@Lexord.de
⚠️ WICHTIG: Speichere diese Keys jetzt sicher (z. B. in einem Passwort-Manager). Wenn du diese Seite schließt, sind sie weg — und beim erneuten Generieren bekommst du andere Keys, mit denen vorher abonnierte Geräte nicht mehr funktionieren.

📋 NÄCHSTE SCHRITTE

  1. Cloudflare → Workers & Pages → lexord-api antippen
  2. Settings → Variables and Secrets → + Add
  3. 3 Secrets anlegen (Type: Secret):
    • • VAPID_PUBLIC_KEY = (siehe oben)
    • • VAPID_PRIVATE_KEY = (siehe oben)
    • • VAPID_SUBJECT = mailto:Kontakt@Lexord.de
  4. Save and Deploy klicken
  5. Admin-Panel öffnen → 🔔 ON → Permission „Erlauben"
  6. 📲 TEST drücken → Notification erscheint
